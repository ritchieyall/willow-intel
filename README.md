# Intelligence Repository Structure

## Directories
- `briefs/`: Raw intelligence briefs (Markdown).
- `categories/`: Categorized summaries and thematic deep-dives.
- `archive/`: Older briefs moved for long-term storage.

## Naming Convention
`YYYY-MM-DD_HHMM_topic-slug.md`

## Metadata Standard (Frontmatter)
Every brief MUST contain:
---
date: YYYY-MM-DD
time: HH:MM
topic: [Topic Name]
category: [Strategic|Tactical|Technical|Regulatory|Personal]
tags: [tag1, tag2]
importance: [Low|Medium|High|Critical]
---
